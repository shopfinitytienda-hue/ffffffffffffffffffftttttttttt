# Tienda Azul Directa

Plantilla WordPress + WooCommerce moderna y adaptable para venta directa sin carrito.

## Instalación

1. Copia esta carpeta en `wp-content/themes/tienda-azul-directa`.
2. Activa el tema desde **Apariencia > Temas**.
3. Instala y activa WooCommerce.
4. Configura las páginas de WooCommerce: tienda, mi cuenta y finalizar compra.
5. Activa el método **Pago contra entrega** y, si aplica, una zona de **Envío gratis**.

## Características

- Encabezado fijo semitransparente con logo, menú, acceso a cuenta y pago.
- Home con slider automático de 3 productos destacados.
- Oferta relámpago con conteo y carrusel de productos en oferta.
- Catálogo adaptable: dos columnas en móvil y cuadrícula fluida en escritorio.
- Producto individual con galería, vídeo opcional, descripción, detalles, variantes, envío gratis y botón de compra directa.
- Sin carrito visible: añadir al carrito redirige al checkout y la página de carrito redirige a pago.
- Checkout moderno, cuenta de usuario para iniciar sesión/registro, ver pedidos, actualizar datos y guardar dirección.
- Recomendaciones tras compra exitosa.

## Vídeo de producto

Agrega una URL de vídeo en el meta campo personalizado `_tad_product_video_url` para mostrarlo en la página del producto.
